# gguf.go

> **Проект на начальном этапе разработки.**
> **Это экспериментальный проект по запуску моделей в формате GGUF на языке Go.**

## О проекте

`gguf.go` - экспериментальная реализация загрузки и выполнения моделей машинного обучения в формате **GGUF** (формат файлов, используемый в экосистеме `llama.cpp`) на чистом **Go**.

Проект создан для изучения низкоуровневой работы с тензорами, форматом GGUF и возможностей Go в задачах инференса ML-моделей без привязки к нативным библиотекам (llama.cpp и тд).

---

Сейчас реализована загрузка файлов GGUF: чтение метаданных и байтов тензоров.

Inference пока не реализован.

Поддерживаются версии GGUF 2 и 3.

```
https://huggingface.co/Qwen/Qwen3-0.6B-GGUF?show_file_info=Qwen3-0.6B-Q8_0.gguf
```

```bash
go build -o build/gguf ./cmd/gguf

# Краткая информация о модели
./build/gguf info -m ./model/Qwen3-0.6B-Q8_0.gguf
# Просмотр метаданных и тензоров
./build/gguf inspect ./model/Qwen3-0.6B-Q8_0.gguf
```
